# Proyecto Festival de Rock

1. Clonar repositorio
2. Instalar dependencias
```
npm install
```

3. Modo desarrollo
```
npm run dev
```

## Dependencias

- SASS
```
npm i -D sass
```

- GULP
```
npm i -D gulp
```

- GULP SASS
```
npm i -D gulp-sass
```

- GULP PLUMBER. Nos ayuda a mantener el watch activo cuando existan errores.
```
npm i -D gulp-plumber
```
